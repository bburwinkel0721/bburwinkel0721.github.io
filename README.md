# SAT Question of the Day 🎓

A lightweight, interactive web application designed to help students practice SAT Math questions. This project was built with **Westerville City Schools** branding and features a self-contained, interactive quiz environment packed with useful study tools.

## 📖 Description

The "SAT Question of the Day" app provides an intuitive interface for students to practice a built-in bank of SAT-style math problems. It supports both multiple-choice and free-response questions and renders complex mathematical formulas beautifully using KaTeX. 

Because it is bundled entirely into a single HTML file (HTML, CSS, and JavaScript), it is incredibly easy to host, share, and use offline.

## ✨ Features

- **Math Formatting**: Uses KaTeX to render inline (`\(...\)`) and block (`$$...$$`) mathematical equations.
- **Interactive Timer**: A built-in countdown timer (defaults to 5:00) to help students practice time management. Includes play, pause, and reset controls.
- **Topic Filtering**: Easily sort the question bank by specific SAT Math topics such as:
  - Algebra
  - Functions
  - Systems
  - Geometry
  - Data & Stats
  - Exponents
  - Linear/Quadratic Equations
- **Instant Feedback**: 
  - Dynamic score tracking (Correct / Total).
  - Immediate visual feedback on correct/incorrect answers.
  - Detailed explanation panels that reveal step-by-step solutions.
- **Question Navigation**: Includes Next/Prev buttons, a Random question generator, and a visual "dots" navigation bar to track attempted questions.
- **Responsive Design**: The UI is optimized for both desktop browsers and mobile devices.

## 🚀 Usage

Since the application is contained entirely within a single HTML file, no build process or server setup is required.

1. **Download** the `index.html` file.
2. **Open** the file directly in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Ensure you have an active internet connection the first time you load it so the browser can fetch the KaTeX library and Google Fonts (DM Sans and Space Mono).
